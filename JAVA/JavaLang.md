## 자바 소개
---
C , C++ 기반을 둠
제임스 고슬링 (James Gosling)이 만듬
원래는 오크(oak)라는 이름 이었음
초기에는 가전제품 탑재용으로 개발이 되었으나. 현재는 인터넷 스마트폰 어플리케이션 서버 애플리케이션을 개발하는 대표적인 언어가 되었다.

## 자바의 특징
---
### 1. 플랫폼에 독립적
- 플랫폼은 기본적으로 하드웨어 운영체제 컴파일러의 집합체 등으로 정의 가능
- C는 컴파일하여 그 플랫폼에서만 실행 가능하다.
- 자바는 어떤 플랫폼에서 컴파일 해도 어디서든 실행 가능하다.
- 이는 인터프리터, 자바가상머신을 이용해 바이트 코드를 읽고 해석해 필요한 작업을 실행하기 때문이다.

### 2. 객체지향
- 객체를 만들고 조립하는 방식의 프로그래밍 기법 
- 캡슐화 상속성 다형성 등이 있음

### 3. Type Safe
- 한 종류의 객체에 양립할 수 없는 다른 타입의 객체를 할당하는 것은 불가능하다.
- 컴파일 에러를 발생시키거나 실행시점 검증해 런타임 에러를 발생시킨다.
  
### 4. 멀티 쓰레딩
- 자바는 멀티 쓰레딩이 가능하다.
  
### 5. 동적 설계 
- 자바는 동적으로 설계 되어있다 
- 라이브러리 등을 동적으로 참조한다.
